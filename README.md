# Setting up

1. Create virtual environment

`python3 -m venv venv`

2. Activate it

`source venv/bin/activate`

3. Install requirements from ***requirements.txt***

`pip install -r requirements.txt`

4. Create *.env* file 

`touch .env`

5. Fill the data like in *.env.template*
